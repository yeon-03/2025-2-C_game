# Speed-Flip (Graphical Version)

**Speed-Flip**은 SDL2 라이브러리를 사용하여 개발된 그래픽 기반의 1:1 카드 매칭 게임입니다.
플레이어와 컴퓨터(AI)가 각자의 보드에서 동시에 게임을 진행하며, 먼저 모든 카드를 매칭하거나 제한 시간 내에 더 많은 카드를 매칭한 쪽이 승리합니다.

## 주요 특징

*   **그래픽 UI (SDL2)**: 직관적인 그래픽 인터페이스와 부드러운 카드 뒤집기 애니메이션
*   **사운드 효과**: 배경음악(BGM), 카드 뒤집기, 매칭 성공/실패, 승리/패배 등 다양한 효과음 제공
*   **실시간 대결**: 플레이어와 AI가 동시에 진행하는 긴장감 넘치는 대결
*   **난이도 시스템**: Easy, Normal, Hard 3단계 난이도 (AI의 정확도와 반응 속도 변화)
*   **콤보 시스템**: 연속 매칭 성공 시 콤보 카운트 증가 및 화면 표시
*   **랭킹 시스템**: 닉네임 입력을 통해 자신의 기록을 남기고, 난이도별 TOP 10 랭킹을 그래픽 화면으로 확인
*   **오프닝 리빌**: 게임 시작 시 3초간 모든 카드를 보여주는 암기 시간 제공

## 실행 화면 및 게임 흐름

1.  **타이틀 화면**: 난이도(Easy, Normal, Hard)를 마우스로 클릭하여 선택합니다.
2.  **닉네임 입력**: 게임에 사용할 닉네임을 입력합니다. (영문/숫자 권장)
3.  **오프닝**: 게임 시작 직후 3초 동안 모든 카드가 공개됩니다. 이때 카드의 위치를 기억해야 합니다.
4.  **게임 플레이**:
    *   마우스 클릭으로 카드를 뒤집습니다.
    *   같은 그림의 카드를 두 장 뒤집으면 매칭에 성공하며 카드가 화면에 고정됩니다.
    *   다른 그림을 뒤집으면 잠시 후 다시 뒷면으로 돌아갑니다.
    *   상단 바를 통해 남은 시간을 확인할 수 있습니다.
5.  **결과 및 랭킹**:
    *   승리/패배 여부가 화면에 표시됩니다.
    *   승리 시 닉네임과 함께 기록이 저장됩니다.
    *   게임 종료 후 그래픽으로 구현된 난이도별 랭킹 화면이 나타나며, 상위 기록을 확인할 수 있습니다.

## 디렉터리 구조

```
speed-flip/
├─ README.md
├─ Makefile             // 빌드 스크립트 (macOS/Linux/Windows)
├─ assets/              // 게임 리소스
│  ├─ images/           // 카드, 타이틀, 승패 이미지 등
│  └─ sounds/           // BGM 및 효과음
├─ data/
│  ├─ ranking.csv       // 랭킹 데이터 (자동 생성)
│  └─ config.json       // 게임 설정 (자동 생성)
├─ src/
│  ├─ main.c            // 메인 진입점, SDL 초기화
│  ├─ game.c / game.h   // 게임 루프, 상태 관리
│  ├─ board.c / board.h // 보드 로직
│  ├─ logic.c / logic.h // AI 로직
│  ├─ io.c / io.h       // 설정 및 랭킹 파일 입출력
│  ├─ render.c / render.h // SDL 렌더링 (커스텀 텍스트 렌더러 포함)
│  ├─ audio.c / audio.h   // 사운드 시스템
│  ├─ util.c / util.h     // 유틸리티 함수
│  └─ common.h          // 공통 상수 및 구조체
└─ build/               // 빌드 결과물
```

## 빌드 및 실행

### 요구 사항

이 게임을 빌드하고 실행하려면 다음 라이브러리가 설치되어 있어야 합니다.

*   **SDL2**
*   **SDL2_image**
*   **SDL2_mixer**
*   **SDL2_ttf** (Makefile에 포함됨)

### 빌드 명령어

터미널에서 프로젝트 루트 디렉터리로 이동한 후 다음 명령어를 입력하세요.

**macOS / Linux**
```bash
# 빌드
make

# 실행
make run

# 정리 (빌드 파일 삭제)
make clean
```

**Windows (MSYS2 MinGW 64-bit)**
```bash
# 빌드
make

# 실행
make run
```

> **참고**: `make run` 실행 시 `assets` 폴더가 `build` 디렉터리로 복사되어 실행됩니다.

## 조작 방법

*   **마우스 왼쪽 클릭**:
    *   난이도 선택
    *   카드 뒤집기
    *   랭킹 화면 넘기기
*   **ESC 키**: 게임 종료

## 설정 (`data/config.json`)

게임 실행 시 `data/config.json` 파일이 없다면 기본값으로 자동 생성됩니다. 이 파일을 수정하여 AI의 난이도 밸런스를 조절할 수 있습니다.

```json
{
   "easy_accuracy"      : 0.55,  // 쉬움 난이도 AI 정확도 (0.0 ~ 1.0)
   "easy_reaction_ms"   : 800,   // 쉬움 난이도 AI 반응 속도 (ms)
   "normal_accuracy"    : 0.75,
   "normal_reaction_ms" : 550,
   "hard_accuracy"      : 0.90,
   "hard_reaction_ms"   : 350
}
```
